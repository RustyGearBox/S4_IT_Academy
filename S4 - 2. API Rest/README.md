# API REST

A simple Java Spring API that performs CRUD with:

- Level 1: H2 Database
- Level 2: MySQL Database

The applications also include logging and Swagger documentation.

## Technologies:
- Java 23
- Spring boot 3.4.2
- Gradle

## Dependencies:
- Spring Boot DevTools
- Spring Web
- Spring Data JPA 
- Lombok
- H2 (Level 1)
- MySQL Driver (Level 2)

## To run a project:

Move to the base folder of the project you want to run and:

- **Build the project:**
  ```sh
  ./gradlew build
  ```
- **Clean the project:**
  ```sh
  ./gradlew clean
  ```
- **Run the application:**
  ```sh
  ./gradlew bootRun
  ```
